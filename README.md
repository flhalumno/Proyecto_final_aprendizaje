# Proyecto_final_aprendizaje

En este proyecto, se usa el data frame Loan Approval Classification Dataset  (https://www.kaggle.com/datasets/taweilo/loan-approval-classification-data), con el objetivo de crear dos modelos, uno de regresión logística y el otro de random forest, seleccionar las variables significativas mediante Recursive Elimination of Features using Cross Validation (REFCV), despues, utilizar algoritmos genéticos para la busqueda de hiperparámetros, y finalmente determinar cual de los dos modelos es mejor.

Si se quiere usar las variables e hiperparametros que ya he calculado y vienen en las carpetas de este repositorio, se debe de eliminar su terminación _f1
